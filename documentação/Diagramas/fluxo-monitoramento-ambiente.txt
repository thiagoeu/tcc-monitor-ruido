== Ciclo de Captura (Loop Infinito) ==

Sensor -> RS485: Envia sinal acústico
activate RS485
RS485 -> Server: Dados de pressão sonora (RFH-01)
deactivate RS485

activate Server
Server -> Server: Valida Checksum (RFH-09)
Server -> DB: Salva medição (RFH-07)
activate DB
DB --> Server: OK
deactivate DB

== Atualização da Interface ==

Server -> Web: Envia dados via WebSocket/API (RFH-06)
activate Web
Web -> Web: Processa design responsivo (RNF-04)
Web --> G: Exibe nível de ruído em tempo real (Latência < 2s)
deactivate Web
deactivate Server
